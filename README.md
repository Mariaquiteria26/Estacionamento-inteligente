## Descrição

Este projeto consiste em um sistema de estacionamento inteligente desenvolvido utilizando Arduino Uno e Tinkercad.

O sistema utiliza sensores ultrassônicos para detectar veículos nas vagas, LEDs NeoPixel para indicar vagas livres e ocupadas e um display LCD para exibir informações ao usuário.

Além disso, o sistema realiza uma cobrança automática baseada no tempo de permanência do veículo, utilizando uma tarifa de R$ 1,00 por minuto.

## Funcionalidades

* Identificação automática de veículos.
* Monitoramento de vagas em tempo real.
* Indicação visual por LEDs.
* Exibição de informações no display LCD.
* Cálculo automático da cobrança.

## Componentes Utilizados

* Arduino Uno
* 3 Sensores Ultrassônicos
* Display LCD 16x2
* LEDs NeoPixel
* Protoboard
* Jumpers
